# Trip Trekker 🗺️

Trip Trekker is a smart trip-planning web application that helps users build efficient, activity-aware itineraries by evaluating multiple travel paths and trip constraints.

## Features

- **Time-Efficient Itinerary Planning** — Suggests optimized travel routes by comparing multiple possible paths.
- **Activity-Aware Trip Design** — Helps users plan trips around activities, hotels, weather, and budget preferences.
- **Modular Travel Dashboard** — Includes dedicated sections for traffic, activities, hotels, emergency info, weather, packing, and budget management.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Routing & UI:** React Router, Framer Motion, Lucide Icons
- **Core Logic:** C++

## How It Works

1. The app presents a clean, multi-section interface for different parts of trip planning.
2. Users can explore traffic, activities, hotels, weather, packing, and budget options in separate views.
3. The system combines trip data and routing logic to support smarter travel decisions.
